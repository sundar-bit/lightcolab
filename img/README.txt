LIGHT COLAB — /img
==================

Drop project & studio images here, then reference them from data/projects.js
(field: image) or directly in index.html.

NAMING (suggested)
  <project-slug>-hero.jpg     e.g. taj-falaknuma-hero.jpg
  team-<firstname>.jpg        e.g. team-asha.jpg

IMAGE GUIDANCE
  - Gallery tiles render at 4:5 (portrait). Featured rows render at 5:4.
  - Export web-sized JPGs (long edge ~1600px, ~70-80% quality) for fast load.
    Original high-res from the Drive folder is fine to keep, but link the
    web-sized copy from the site.
  - Always meaningful, descriptive filenames — they become part of alt text logic.

BRAND / META ASSETS (placeholders referenced in index.html <head>)
  favicon.ico            multi-size .ico
  favicon.svg            scalable favicon
  apple-touch-icon.png   180x180
  og-image.jpg           1200x630 social share image

This README is just a note; it has no effect on the site and can be deleted.
